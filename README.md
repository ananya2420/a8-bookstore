Project Name: Book Store

Purpose: The purpose of this project is to develop a full-stack book management web application where users can explore, search, and view details of various books in an organized and interactive way. The platform allows users to filter books by category, search by title, and navigate through dynamic routes to access individual book details. A key objective of the project is to implement authentication, ensuring that only logged-in users can perform specific actions such as borrowing books and accessing protected pages like their profile. It also demonstrates how to manage user sessions, handle conditional rendering based on authentication status, and provide a smooth user experience with features like toast notifications and responsive design. Overall, the project showcases the integration of frontend and backend technologies to build a modern, user-focused application with real-world functionality.

key features: The key features of this project include a dynamic and user-friendly interface that allows users to browse and explore a collection of books with ease. Users can view featured books on the home page and navigate to detailed pages for each book using dynamic routing. The application includes a powerful search and filtering system, enabling users to find books by title and category such as Story, Tech, and Science. Authentication is a major feature, allowing users to register, log in (including Google social login), and access protected routes. Only authenticated users can borrow books, ensuring proper access control. The project also provides real-time feedback through toast notifications for actions like login, errors, and successful borrowing. Additionally, users have a personal profile page where they can view and update their information, with data securely stored in a database. The design is fully responsive and visually appealing, built using Tailwind CSS and DaisyUI, and enhanced with interactive elements for a smooth user experience.
